## Sub-phases

### 7.1 — `match_movie` true chain iteration (DESIGN §6.2 verbatim)

Replace:

```python
tmdb_client = cast("TMDBClient", self._registry.get("tmdb"))
try:
    candidates = tmdb_client.search_movie(title, year=year)
    ...
except Exception as e:
    result.error = ...
```

With:

```python
providers = self._registry.chain(MovieDetailsProvider)
attempted: list[AttemptOutcome] = []
match: ProviderMatch | None = None

for provider in providers:
    try:
        candidates = provider.search_movie(title, year=year) if hasattr(provider, "search_movie") else provider.search(title)
        if not candidates:
            attempted.append(AttemptOutcome(RegistryProviderName(provider.provider_name), "empty_result"))
            log.debug("registry_provider_skip", provider=provider.provider_name,
                      capability="MovieDetailsProvider", reason="empty_result")
            self._registry._emit_provider_fallback(
                capability="MovieDetailsProvider",
                from_provider=provider.provider_name,
                reason="empty_result",
                item={"title": title, "year": year},
            )
            continue
        # ... (best_match logic)
        match = ProviderMatch(provider=RegistryProviderName(provider.provider_name), id=chosen.id, media_type=MediaType.MOVIE)
        break
    except CircuitOpenError:
        attempted.append(AttemptOutcome(RegistryProviderName(provider.provider_name), "circuit_open"))
        log.debug("registry_provider_skip", provider=provider.provider_name, capability="MovieDetailsProvider", reason="circuit_open")
        continue
    except NetworkError as e:
        attempted.append(AttemptOutcome(RegistryProviderName(provider.provider_name), "network", detail=type(e).__name__))
        log.warning("registry_provider_fail", provider=provider.provider_name, capability="MovieDetailsProvider", exc_type=type(e).__name__)
        continue

if match is None:
    self._registry._emit_provider_exhausted(
        capability="MovieDetailsProvider",
        attempted=attempted,
        item={"title": title, "year": year},
    )
    log.error("registry_chain_exhausted", capability="MovieDetailsProvider", attempted=[(a.provider, a.reason) for a in attempted], item={"title": title, "year": year})
    raise ProviderExhausted(MovieDetailsProvider, attempted=attempted, item_context={"title": title, "year": year})
```

The two `self._registry._emit_*` are new public helpers on `ProviderRegistry` that wrap `_event_bus_safe_emit` with the proper event dataclass.

Update mocks in `tests/scraper/test_movie_service*.py` and `tests/integration/scraper/`.

Commit: `feat(registry): movie_service true chain iteration with empty_result/circuit_open/network branches`

### 7.3 — Add `ProviderRegistry._emit_provider_fallback` + `_emit_provider_exhausted` helpers

Two public-on-instance helpers that construct the event dataclass + call `_event_bus_safe_emit`. Keeps emission logic centralized so future consumers don't re-implement.

```python
def _emit_provider_fallback(
    self,
    *,
    capability: str,
    from_provider: str,
    reason: Literal["circuit_open", "network", "empty_result"],
    item: dict[str, Any],
    to_provider: str | None = None,
    exc_type: str | None = None,
) -> None:
    self._event_bus_safe_emit(
        ProviderFallbackTriggered(
            capability=capability,
            from_provider=from_provider,
            to_provider=to_provider or "",
            reason=reason,
            exc_type=exc_type,
            item=item,
        )
    )

def _emit_provider_exhausted(
    self,
    *,
    capability: str,
    attempted: list[AttemptOutcome],
    item: dict[str, Any],
) -> None:
    self._event_bus_safe_emit(
        ProviderExhaustedEvent(
            capability=capability,
            attempted=attempted,
            item=item,
        )
    )
```

Add unit tests asserting both helpers emit the right event class with correct fields.

Commit: `feat(registry): public emit helpers for chain fallback + exhausted events`

```python
def fan_out[C: FanOutCapability](self, capability: type[C]) -> FanOutResult[C]:
    ...
    return FanOutResult(values=eligible, attempted=attempted)
```

Update the (currently zero) production consumers + the unit tests. The `RegistryFanOutCompleted` event still fires as before — the dataclass return is for callers who want sync access to provenance without subscribing to the bus.
